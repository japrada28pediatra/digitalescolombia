# Directorio odontológico de Colombia

Paquete estático preparado para publicarse en:

`https://digitalescolombia.com/directorio-odontologico/`

## Publicación

Subir **todo el contenido de esta carpeta** dentro de `public_html/directorio-odontologico/` o de la carpeta equivalente del hosting. No subir la carpeta contenedora como una carpeta adicional.

Incluye portada con buscador y filtros, páginas por ciudad, páginas individuales, datos estructurados, sitemap.xml, robots.txt y reglas `.htaccess`.

La base es un registro público de descubrimiento. Los datos de contacto y la actividad actual de cada profesional deben verificarse antes de publicarlos como recomendación comercial.
