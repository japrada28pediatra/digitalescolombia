- Home completa orientada a conversión.
- Diseño responsive.
- Menú mobile.
- Botón flotante de WhatsApp.
- Barra sticky en mobile.
- Formulario que genera mensaje de WhatsApp.
- Página de servicios.

  - Laureles

- `573001234567` por el número WhatsApp con indicativo, sin espacios.

## Recomendación SEO

Para publicar mejor:

1. Configurar dominio real en canonical y sitemap.
2. Crear Google Business Profile si el negocio tiene operación real.
3. Agregar fotos reales del técnico, vehículo, herramientas o trabajos.
4. No usar logos de marcas como Samsung, LG, Haceb, Whirlpool, Mabe, etc., salvo autorización.
5. Si se mencionan marcas, aclarar: “servicio técnico independiente multimarcas”.
6. Revisar que las garantías, precios y cobertura sean reales.
7. Conectar eventos de clic en WhatsApp, llamada y formulario para Google Ads / GA4.
