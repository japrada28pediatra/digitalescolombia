# Directorio Abogados Colombia

Sitio estático generado con 602 perfiles únicos a partir de 980 filas.

## Antes de publicar
1. Reemplazar `https://tudominio.com` en todos los archivos, robots.txt y sitemap.xml.
2. Configurar el formulario de registro.
3. Revisar perfiles pendientes antes de promocionarlos como verificados.
4. Subir el contenido de esta carpeta a public_html.
5. Enviar sitemap.xml a Google Search Console.

## Archivos principales
- index.html
- directorio.html
- firmas/*.html
- ciudades/*.html
- assets/data.json
- directorio.csv
- sitemap.xml
- robots.txt
