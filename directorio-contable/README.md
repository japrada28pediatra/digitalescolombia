# Directorio Contable Colombia

Sitio estático generado con 825 perfiles.

## Antes de publicar
1. Reemplazar `https://directoriocontablecolombia.com` por el dominio definitivo en todos los archivos.
2. Revisar y verificar fichas antes de indexarlas masivamente.
3. Conectar el formulario de registro.
4. Añadir GA4, Search Console y medición de conversiones.
5. Subir todo el contenido a `public_html`.
6. Enviar `/sitemap.xml` a Google Search Console.

## Arquitectura
- Inicio con buscador y filtros.
- Landing SEO de Bogotá.
- 825 páginas individuales.
- 4 páginas de servicios.
- 3 artículos iniciales.
- Schema JSON-LD, canonical, Open Graph, robots.txt y sitemap.
